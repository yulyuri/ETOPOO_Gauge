# Etopoo Digital Gauge Monitor

Web-based monitoring tool for Etopoo digital dial gauges using Web Serial API.

[Live Demo](https://gauge.tanscloud.space) | [Documentation](docs/protocol.md)

## Features
- Real-time measurement display
- Tolerance checking (USL/LSL/STD)
- Automatic button press detection
- Data logging and CSV export
- Statistics tracking (min/max/avg/range)
- Zero/offset functionality

## Start
1. Open `index.html` in Chrome/Edge (desktop)
2. Connect your Etopoo gauge via USB
3. Click "Connect to Serial Port"

## Browser Requirements
- Chrome/Edge desktop (Web Serial API support.)
- HTTPS required (or localhost for testing)
- Most likely will not work on A-star laptop due to restrictions
